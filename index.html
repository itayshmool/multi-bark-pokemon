// MOVE SET

FEATURES

Six powerful moves your pup team comes equipped with right out of the Pokéball.

ELECTRIC ⚡
MULTI-BACKEND

Four LLM backends in one pack: Claude Code, Cursor, OpenAI Codex, and Gemini. Tag #claude-code or #gemini to pick your fighter.

PSYCHIC 🔮
SAVE STATE

Persistent session memory across restarts. Each pup remembers your conversation. /reset to wipe, /reborn to revive.

NORMAL 🎙️
VOICE INPUT

Send a voice note and whisper.cpp transcribes it locally — 90 languages supported, zero cloud, zero data leakage.

FIRE 🔥
REVIVE

Auto-fallback with exponential backoff. Crashes, context limits, and rate limits trigger automatic recovery with context injected from rolling summaries.

WATER 📦
ITEM TRADE

Pups drop files into their outbox and the server delivers them straight to your chat. Images, patches, logs — anything goes.

DRAGON 🐉
TEAM BATTLE

Pups delegate to sub-agents via bark delegate. Add --branch for isolated git branches and automatic PRs.

COMMANDS

Every pup in your party responds to these commands from any connected chat platform.

/status Refresh the pinned status message — see all pups and their current activity.
/create Reply to any message to spawn a new pup with that context. Add instructions or @name.
/stop name Send Ctrl+C to a running pup mid-task. Use pack to stop all.
/clear name Shelve a pup (deactivate, preserves history). Restore later with /reborn.
/reset name Wipe a pup's memory and start fresh — same pup, new session, same backend.
/reborn name Resurrect a shelved pup with its full session history intact.
/daily Request a one-line standup update from every active pup in the pack.
/stats Show cost and token usage summary — per backend and per pup. Add a name for details.
/skills List available skills. Use /skill name @pup to equip a skill on a pup.
/backends Show all available LLM backends and their capability flags.

GET STARTED

From zero to pup army in four steps. A Pokémon trainer's first steps into the wild.

1

CLONE THE REPO

Grab multi-bark-pack from GitHub and enter the directory.

git clone https://github.com/itayshmool/multi-bark-pack.git cd multi-bark-pack
2

INSTALL PREREQUISITES

The installer checks and sets up Node.js, tmux, yarn, Claude CLI, ffmpeg, and whisper.cpp automatically.

bash prerequisites.sh
3

CONFIGURE YOUR .ENV

Copy the example env and fill in your platform tokens — WhatsApp, Telegram, or Slack (or all three).

cp .env.example .env # Add your tokens: # WA_ENABLED=true # TELEGRAM_TOKEN=your-bot-token # SLACK_BOT_TOKEN=xoxb-...
4

START THE SERVER

Fire it up. The server auto-restarts on clean exit. Send a message to your bot to spawn your first pup.

yarn start # → Server online! Send a WhatsApp/Telegram/Slack message. # → Your first pup (Pikachu ⚡) is ready.
